HTML

Html no es un lenguaje de programaciòn por si mismo, pero se lo considera un lenguaje de marcado de hipertexto(html), sirve para estructurar el contenido de una página web, es aquello que le da la forma y orden a la página, se podría decir que es su esqueleto.

Este se maneja con diversas etiquetas, para agregar diversos contenidos, como títulos, párrafos, enlaces, imágenes, o videos.

CSS

CSS es el único lenguaje de diseño y lo interpreta el navegador sin hacer falta instalar nada. En este los estilos que están por debajo pisan a los archivos por encima por ejemplo si declaramos un estilo en la línea 20 y otro en la 24 se toma el último ya que es el de más importancia
