Sintaxis CSS

selector : el elemento que se le aplica estilo

propiedad:lo que se va a cambiar, no hay límite de propiedades y el (;) es para separar las propiedades y es obligatorio.

valor: el valor que se le da a la propiedad

propiedad:valor-> es una declaración

Selectores

    Selectores Simples

  • selector universal: se usa para cosas concretas que engloban todo o seleccionan todos los elementos (*)
  • selector de etiqueta: nombre de la etiqueta
  • id: identidad del elemento.
  • clase: clase del elemento, no tiene limites de numeros de clase, lo bueno es que con una clase dar estilo a varios elementos y reutilizar el codigo
    • Selectores Compuestos
  • selectores decendientes: se puede seleccionar un elemento dentro del otro, no es bueno bajar mas de dos elementos, el elemento dentro del otro se le llama hijo y el que lo contiene padre
  • selectores de hermano siguiente(+)(~)

Propiedad Margin

nos permite agregar margenes entre elementos y tiene 4 valores nada mas

  • margin-top: Margen superior
  • margin-left: Margen izquierdo
  • margin-right: Margen derecho
  • margin-top: Margen inferior

Etiquetas

Html se maneja mediante etiquetas, estas sirven para diversas funciones.Algunos ejemplos de estas etiquetas son:

H : sirven para agregar un título, estos siguen un orden jerárquico, yendo del h1, que es el título más importante, descendiendo en una escala de importancia hasta el h6.

p : Sirve para agregar párrafos que contienen texto.

a : Sirve para agregar un enlace, ya sea a otro documento, a otra página web, o a una hoja de estilos o de script.
