Estructura

-Head:Se declara la información sobre el documento, vease el titulo, enlaces a scripts y hojas de estilos.

-Body: Se declara todo el contenido de la página web.

-Footer: Es el pie de página, suele contener información sobre el autor y los creadores y la bibliografía o documentos relacionados.

-Section: Es utilizado para agrupar contenidos relacionados

- Article: Es para agregar información de otro sitio y que no pierda el sentido

-Aside: Es para contenido que no es de la página, como por ejemplo anuncios

Etiquetas

Html se maneja mediante etiquetas, estas sirven para diversas funciones.Algunos ejemplos de estas etiquetas son:

H : sirven para agregar un título, estos siguen un orden jerárquico, yendo del h1, que es el título más importante, descendiendo en una escala de importancia hasta el h6.

p : Sirve para agregar párrafos que contienen texto.

a : Sirve para agregar un enlace, ya sea a otro documento, a otra página web, o a una hoja de estilos o de script.

Tablas

En la estructura de la tabla podemos encontrar las siguientes etiquetas:

-Table: encierra una tabla

-tr(Table Row): etiqueta que forma una fila

-td(Table Data): etiqueta que forma una celda

-Caption: Sirve para poder agregarle un título a la tabla

-Tfoot: sirve de forma opcional para poder agregar algo al pie de la tabla

Listas

Se puede utilizar listas ordenadas "ol", o listas desordenadas "ul", y para declarar los elementos que irán dentro de la listas se utiliza la etiqueta li.
